What are Networks?

A Network or graph is a collection of points called nodes connected by lines called vertices. They are incredibly usefukk tools for modelising interconnected entities such as cities or people.

What can I do with graphs?

Using graphs we can find the shortest path between two objects, using a wide veriaty of algorithms:

Breadth First Seach
An alogorith that finds the shortest path between two nodes by radiating out from the starting node
Dikstra's algorithm
Used in a weihgted graph, that is one in which each vertex has a weight. It uses this weighting to determine to prioritise nodes closer to the start
A* Algorith
Based on Djikstra's algorithms but takes into account the distance between a point and the end in determining which node to visit next
